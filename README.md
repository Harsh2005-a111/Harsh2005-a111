<summary><b>🧠 NeuroFusionNet — Brain Tumor MRI Classification Ensemble</b></summary>

**Multi-Model Learned Ensemble for 4-Class Brain Tumor MRI Classification**

Designed a production-grade deep learning ensemble combining three specialized networks with novel custom Keras modules for clinically-significant brain tumor classification.

| Metric | Value |
|--------|-------|
| **Accuracy (Ensemble)** | 96.56% (Neural Stacker) |
| **No Tumour Recall** | 100% (Zero False Negatives) |
| **Pituitary Recall** | 99.75% (Clinically Critical) |
| **Dataset Scale** | 7,200 MRI images |
| **Architecture Innovation** | 3 Novel Keras Modules (CEA, DRP, BFH) |
| **Baseline Comparison** | +1.93% improvement over single-model (94.63%) |
| **Publication Status** | IEEE-style Research Paper (In Review) |
| **Deployment** | Hugging Face Spaces (Integrated in NeuroFusionNet App) |
| **Repository** | [NeuroFusionNet HF Space](https://huggingface.co/spaces/Harsh8-6-2005/multimodal_speech_brain_tumor_app/) |

**Key Architectural Innovations:**
- **CEA (Contextual Entropy Attention)**: Variance-based feature weighting with learnable scalar α, applied at 5 architectural points
- **DRP (Dilated Residual Pyramid)**: 4 parallel dilated convolutions (rates 1/2/4/8) for multi-scale receptive fields
- **BFH (Bidirectional Feature Harmonization)**: Cross-stream mutual gating for semantic feature extraction

**Technical Details:**
- Two-phase transfer learning (frozen backbone 20 epochs + top-layer fine-tuning 30 epochs)
- Label smoothing (ε=0.10) with deep supervision
- Multi-output training for clinically-weighted loss functions
- Resolved memory constraints and gradient stability issues

**Impact:** Clinically significant high recall on critical tumor classes; production-ready ensemble achieving state-of-the-art performance.

**Cross-Modal Affective Computing System: EEG Signals + Speech Emotion Fusion**

| Component | Performance | Dataset | Details |
|-----------|-------------|---------|---------|
| **Speech Branch (DCNN-DTPM-SE)** | 91.81% WAR | EMO-DB (535 utterances, 7 emotions) | +4.50% over baseline (87.31% Zhang et al. 2018) |
| **DTPM Innovation** | +7% absolute gain | CREMA-D (6,442 utterances) | 68.50% → 75.59% through temporal context modeling |
| **Multimodal Fusion** | Integrated | DEAP EEG + Speech | Cross-modal gated architecture, semantic alignment |
| **Memory Optimization** | 18 MB peak RAM | Kaggle notebook | Resolved 35 GB bottleneck via Keras data generators |
| **Publication Status** | IEEE-style Paper | In Review | Comprehensive clinical-grade documentation |
| **Deployment** | Live Application | Hugging Face Spaces | [NeuroFusionNet Multimodal App](https://harsh8-6-2005-multimodal-speech-brain-tumor-app.hf.space/) |

**Speech Emotion Recognition (DCNN-DTPM-SE):**
- EfficientNetB0 backbone with Squeeze-Excitation (SE) recalibration
- Dynamic Temporal Pyramid Matching (DTPM) for temporal dependency modeling
- 3-channel log-Mel spectrograms: static + delta + delta-delta as (64, T, 3) tensors
- SpecAugment data augmentation with on-the-fly Keras generators

**Multimodal Fusion Architecture:**
- EEG branch: Band-pass filtering, ICA preprocessing, multi-resolution spectral features
- Speech branch: Log-Mel spectrogram extraction with temporal derivatives
- Fusion layer: Cross-modal gating with learned weights
- Output: Joint emotion classification across 6 emotion classes

**Technical Achievements:**
- Exceeded published benchmark (Zhang et al. 2018 IEEE TMM) by 4.50% WAR
- Resolved mixed_float16 NaN gradients in SE sigmoid layers (float32 casting)
- Eliminated 35 GB memory bottleneck through smart data generation
- LOSO (Leave-One-Subject-Out) cross-validation on EMO-DB: 91.81% ± 12.50%

**Status:** ✅ **COMPLETED** — Live on Hugging Face with full inference pipeline.
